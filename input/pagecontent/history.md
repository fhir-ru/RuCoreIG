# История изменений

На данной странице приводятся сведения о версиях Implementation Guide RuCore и о ключевых изменениях между версиями.

## Версия 0.3.0

Статус: draft

Ключевые изменения:

* Пересмотрена интерпретация профиля `Core_Address`.
* `Core_Address` рассматривается как профиль адреса на территории Российской Федерации, а не как универсальный национальный профиль любого адреса.
* Для адресов объектов, расположенных на территории Российской Федерации, применяется профиль `Core_Address`.
* Для адресов вне территории Российской Федерации применяется базовый тип данных `Address`.
* В профилях ресурсов и в проекте ГОСТ начата соответствующая корректировка ссылок и формулировок, связанных с использованием `Core_Address`.

## Версия 0.2.0

Статус: baseline

Ключевые изменения:

* Сформирован исходный набор базовых профилей RuCore на основе FHIR R5.
* Определены базовые ресурсы, расширения, наборы значений и примеры для российского контекста использования.
